# Backend Setup Guide

1. **Install Node.js**: Ensure Node v18+ is installed on your machine.
2. **Clone backend repository**: `git clone https://github.com/company/backend-service.git`
3. **Run local server**: Navigate to the directory, run `npm install`, then `npm run dev`. The server should start on port 3000.
4. **Read backend architecture docs**: Review `architecture.md` in the repo root to understand our microservices.
5. **Review API standards**: We use REST for public APIs and gRPC for internal services.
6. **Complete starter bug fix**: Pick a "good first issue" from the internal Jira board and submit a PR within your first week.
