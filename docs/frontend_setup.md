# Frontend Setup Guide

1. **Install Dependencies**: Make sure you have `yarn` installed globally.
2. **Clone frontend repository**: `git clone https://github.com/company/frontend-web.git`
3. **Run local server**: Navigate to the directory, run `yarn install`, then `yarn start`. The app will launch on port 8080.
4. **Style Guide**: We use TailwindCSS and a custom internal component library called 'CompanyUI'.
5. **Testing**: All components must have 80% test coverage using Jest and React Testing Library.
